A propos d'Uninstall

Désinstalle les programmes qui ont été installés via SoftwareValet.

Comment l'utiliser?

Sélectionnez un programme dans la liste de ceux installés, et cliquez sur le bouton "Désinstaller".

Désinstallation

Oui, vous pouvez maintenant utiliser "Uninstall" pour désinstaller "Uninstall" (It turns out you always could...).
Après avoir désinstallé "Uninstall", vous pouvez continuer à l'utiliser jusqu'à ce que vous le fermiez.

Historique

23 Mars 2001
version 1.3
  • Correction d'un bug dans la liste. Merci à Mickael Wolff pour me l'avoir signalé.
29 Janvier 2001
version 1.2
  • Ajout d'une fenêtre "A propos de...".
  • La documentation peut être appelée directement via le menu "Aide/Documentation" de "Uninstall".
  • Le répertoire des fichiers ".log" peut maintenant être sélectionné via une boîte de dialogue.
  • La localisation de "Uninstall" peut maintenant se faire en traduisant les chaînes de caractères se trouvant dans le fichier "UninstallLanguage.h".
  • Les contrôles sont maintenant placés sur un fond gris (couleur standard des boîtes de dialogues de BeOS).
3 Juillet 2000
version 1.1
  • Correction d'un bug qui empêchait la fenêtre de sélection d'être mise à jour après une installation.
  • Les préférences ont été ajoutées :
    • Maintenant le répertoire des fichiers ".log" peut être choisi.
    • L'utilisateur peut maintenant définir quelles actions seront prises quand un répertoire n'est pas vide.
